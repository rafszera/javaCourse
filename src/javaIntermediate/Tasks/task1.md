Create a Java program to manage ninja information using inheritance. Develop a base class called Ninja and a subclass called Uchiha.

1 - In the Ninja class, define the attributes:
name (String), age (int), mission (String), difficultyLevel (String), and missionStatus (String).
Add a method called showInformation() that displays all these attributes.
2 - Create a subclass called Uchiha that inherits from Ninja.
Add an additional attribute specialAbility (String) and a method called showSpecialAbility() to display the ninja's special ability.
3 - Override the method showInformation() in the Uchiha class to also include the special ability.
